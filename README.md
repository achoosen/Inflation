A Jupyter notebook to extract summary statistics of .xlsx files from:
- https://www.bi.go.id/id/statistik/indikator/data-inflasi.aspx (Data Inflasi.xlsx)
- https://inflationdata.com/articles/current-inflation/#Current_Inflation_Table (Annual Inflation Table.xlsx)
- https://www.clevelandfed.org/indicators-and-data/inflation-expectations (Inflation expectations.xlsx)

# Prerequisite
MUST HAVE the .xlsx files (see above) in the same folder as Inflasi.ipynb
